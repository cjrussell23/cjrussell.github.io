Knight of Xylasia

Knight of Xylasia is a 2D RPG platformer created using Unity and C#. This game is still a work in progress. The main features are all complete. It has a main menu, a tutorial level, a first level, and a boss level. The main gameplay involves the player jumping on platforms while fighting enemies to get to the end of the level. The player has an inventory for the various fruit drops which come from killing enemies. Each fruit has different abilities which can give the player boosts to health, mana, or damage. The player has multiple attacks and mana can boost attacks or movement abilities.
